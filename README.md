# Smart revise automation

## Features

* Uses both qwen3:30b and google AI to determine correct choices on smart revise questions
* Uses pyautogui module to move mouse and click on correct choice
* Uses webbrowser module to search up queries based on question
* Takes screenshots that are translated into text to be fed to the AI model
* Has a + 80% accuracy in answering questions.

## Platform
Only available on Windows for now.

## Build ##
Open the solution via Visual Studio, build and run.
