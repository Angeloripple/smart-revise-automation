# Instructions

## Step 1: Download the ollama model
* Download ollama application at https://ollama.com
* Open command prompt and run "ollama run qwen3:30b"

## Step 2: Download all the python modules
* Open terminal
* pip install ollama
* pip install pyautogui
* pip install mss
* pip install numpy
* pip install time
* pip install requests
* pip install re
* pip install os 
* pip install webbrowser
* pip install keyboard

## Step 3: Open two tabs
* Open two browser tabs:
* One should be open on smart revise questions
* One as an empty browser 
* Ensure they are seperate window tabs so that alt+tab can alternate between the two

## Step 4: Copy source code into visual studio
* Open visual studio and copy past the code ensuring that prior steps have been done

## Step 5: Run code
* Click run on the program
* Quickly switch to the two tabs ensuring that it alternates between the two tabs with alt+tab
* Stay on the Smart revise tab with question and let the magic carry out
